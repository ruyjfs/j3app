Você ainda não possui uma organização, você pode criar uma organização ou pedir para alguém adiciona-lo à outra organização já existem.

É preciso criar uma organização para criar os produtos e suas tarefas, definir as sprints e responsáveis pelas tarefas.

Após criar a organização será possível adicionar os membros da organização e criar as equipes da organização que iram participar dos produtos e realizar as tarefas.

Para saber mais, clique no ícone de interrogação no canto direito superior. Lá explicamos tudo sobre o scrum e o j3scrum.

Para criar uma organização, clique no botão vermelho.
